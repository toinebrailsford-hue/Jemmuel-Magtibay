# Assets

This folder is reserved for local website assets.

The current website still references hosted Squarespace/Spotify/Google assets because this sandbox cannot download remote files. Use `asset-sources.md` as the source map for the images and logos that should be downloaded into:

- `assets/images/`
- `assets/logos/`

After those files are added, update `index.html` and `styles.css` to use the matching local paths.
